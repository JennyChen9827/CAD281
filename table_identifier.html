Identifier Extractor in C – Explained Step by Step

Step Code Snippet Explanation
1. Check Valid Identifier Start is_letter_or_underscore(c) Returns true if c is a letter (A–Z, a–z) or underscore (_)
2. Check Valid Identifier Characters is_letter_digit_or_underscore(c) Returns true if c is a letter, digit (0–9), or underscore (_)
3. Open File fopen("data.txt", "r") Opens file for reading; exits with message if not found
4. Read One Character c = fgetc(fp) Reads one character at a time from the file
5. Skip Invalid Start if (!is_letter_or_underscore(c)) If not a valid start, skip the whole word using a loop
6. Start Valid Identifier word[idx++] = c If it's a valid start, begin saving it into word[]
7. Continue Saving Identifier while (...) { word[idx++] = c; } Continue saving characters as long as they are valid (letters/digits/_)
8. End of Identifier word[idx] = '\0' Add null terminator so it's a proper string
9. Print Identifier printf("Identifier: %s\n", word) Print the valid identifier found in the file
10. Reset for Next Word idx = 0 Reset the index to collect the next word

✅ Example:

Input file:  
@hello 123 _start validName 9test

Output:  
Identifier:  _start  
Identifier: validName
